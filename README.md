The SoftGrowTech repository is a compilation of four distinct web development projects that demonstrate various aspects of front-end development. ## Project 1: Web Music Player

### Description
A fully functional Web Music Player built with vanilla JavaScript, HTML, and CSS. This application provides users with an interactive music listening experience with playlist management and playback controls.

### Features
- **Audio Playback**: Play, pause, skip to next/previous tracks with full playback controls
- **Playlist Management**: Display and navigate through multiple music tracks
- **Progress Bar**: Visual progress indicator with seek functionality
- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- **Visual Feedback**: Interactive UI with hover effects and visual cues for user actions
- **Album Art Display**: Shows album artwork for the currently playing track

### Technologies Used
- **HTML5**: Semantic markup for content structure and audio element
- **CSS3**: Advanced styling with flexbox layout, transitions, and responsive design
- **Vanilla JavaScript**: DOM manipulation, event handling, and audio API integration
- **Responsive Design**: Media queries for multiple device support

### File Structure
```
Project1_Web_Music_Player/
├── index.html          # Main HTML file with player structure
├── style.css           # Comprehensive CSS styling with responsive design
├── script.js           # JavaScript functionality for audio playback
└── assets/             # Music files and album artwork
```

### Key Implementation Details
- **Audio Controls**: Play/pause, next/previous track navigation
- **Progress Tracking**: Real-time progress bar updates and manual seeking
- **Track Information**: Display song title, artist name, and album art
- **Responsive Layout**: Adapts to different screen sizes with mobile-first approach
- **User Experience**: Smooth transitions and visual feedback for user interactions

### Design Philosophy
The Web Music Player follows a modern, clean design philosophy with a focus on usability and visual appeal. The interface is designed to be intuitive for users of all technical backgrounds, with clear playback controls and an attractive visual presentation.

## Project Setup

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Text editor or IDE (VS Code, Sublime Text, etc.)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/username/SoftGrowTech.git
   ```

2. Navigate to the project directory:
   ```bash
   cd SoftGrowTech
   ```

3. Open each project folder separately:
   - For Web Music Player: `cd Project1_Web_Music_Player/`
   - For To-Do List: `cd Task2_To-Do_List_App/`
   - For Resume Website: `cd Task3_Resume_Website_Project/`
   - For Landing Page: `cd Task4_Responsive_Landing_Page/`

4. Open the `index.html` file in your preferred browser, or serve via a local development server for better functionality (especially for the Resume Website which has path issues in the HTML file).

### Running the Projects
Each project is self-contained and can be run directly by opening the `index.html` file in a web browser. For the most accurate experience, particularly with the Resume Website project (which references CSS and JS files with incorrect paths in the current HTML), you may want to temporarily update the file paths in `Task3_Resume_Website_Project/index.html` from `css/style.css`, `js/script.js` to `style.css`, `script.js` respectively, or serve the files through a local web server.

### Local Development Server Options
- Python: `python -m http.server 8000` (Python 3) or `python -m SimpleHTTPServer 8000` (Python 2)
- Node.js: `npx http-server`
- PHP: `php -S localhost:8000`
- Live Server extension in VS Code
